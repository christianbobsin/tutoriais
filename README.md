# Tutoriais

Repositório onde guardo tutoriais e anotações práticas de configuração para ambientes que uso no dia a dia.  
O objetivo é centralizar passos raros, ajustes chatos de lembrar e guias rápidos por plataforma.

## Estrutura

Os tutoriais estão organizados por diretório:

- `hardware/` → ajustes ligados a equipamento e sistema
- `linux/` → guias para ambientes Linux, com foco em Ubuntu e Debian
- `vscode/` → customizações e dicas para Visual Studio Code
- `windows/` → tutoriais para Windows
